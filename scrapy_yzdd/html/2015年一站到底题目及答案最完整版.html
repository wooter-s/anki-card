2015年一站到底题目及答案最完整版

时间：2015-10-30 16:29:55本文内容及图片来源于读者投稿,如有侵权请联系xuexila888@qq.com 欧东艳 我要投稿

　　《一站到底》是江苏卫视继《非诚勿扰》、《非常了得》等一系列金牌栏目后，于2012年3月2日推出的一档全新益智答题类节目，下面是2015年一站到底题目及答案最完整版，供大家参考。

　　1、毛主席是哪一年出生哪一年逝世的D

　　A 1893~1967 B 1896~1976 C 1896~1967 D 1893~1976

　　2、邓小平的故乡在哪D

　　A北京B成都C重庆D广安

　　3、中国第一民办普通本科高校是谁办的C

　　A杨雪梅B杨宝成C胡大白D孔令云

　　4、易中天毕业于B

　　A广西大学B武汉大学C北京大学D西南大学

　　5、首次对斯大林体制进行改革的是谁B

　　A列宁B赫鲁晓夫C戈尔巴乔夫D勃列日涅夫

　　6、以下人物中，谁被称为王佐之才C

　　A周瑜B诸葛亮C荀D郭嘉

　　7、曹操阵营中，谁识破了庞统的C

　　A荀攸B荀C徐庶D程昱

　　8、“江东四杰”是指A

　　A周瑜、鲁肃、吕蒙、陆逊B周瑜、张昭、鲁肃、张继C鲁肃、黄盖、程普、甘宁D周瑜、张昭、甘宁、吕蒙

　　9、梁山排名25位的是B

　　A李俊B雷横C阮小二D张横

　　10、征方腊谁不是李逵的副手C

　　A鲍旭B李衮C邓飞D项冲

　　11、《三国志》作者陈寿的父亲是哪位

　　A陈应B陈武C陈表D陈文

　　12、史进的师傅是A

　　A王进B宋江C林冲D花荣

　　13、沉鱼落雁的“落雁”是指谁

　　A西施B王昭君C貂蝉D褒姒

　　14、下列哪个是出自柳公权的作品

　　A九成宫醴泉铭B麻古仙坦记C玄秘塔碑D醉翁亭记

　　15、文武2庙中亚圣分别是指

　　A张良，关羽B诸葛亮，韩信C孟子，张良D孔子，关羽

　　16、孙权的字是什么

　　A刘备B孟德C仲谋D仲夏

　　17、清高宗是清代哪一皇帝的庙号

　　A顺治B乾隆C雍正D康熙

　　18、明代王守仁创立的学说是指

　　A西学B玄学C理学D心学

　　19、二战时虎式坦克出现后德国又研制出什么

　　A虎王坦克B E-100坦克C虎式P坦克D鼠式坦克

　　20、日本史的“武家政权第一人是”

　　A源赖潮B源赖家C北条政子D平清盛

　　21、世界最大坦克战是指二战时期的

　　A库尔斯克会战B曼施坦因战役C克拉玛依会战D哈尔科夫战役

　　22 、1684年清政府设置的机构是

　　A理藩院B贵州布政使司C澎湖巡检司D台湾府

　　23、“给我20年，还你一个强盛的俄罗斯”是谁说过这句霸气的话

　　A普京B奥巴马C梅德韦杰夫D切尔诺阿尔法

　　24、苏联那一位领导人强调种玉米

　　A列宁B赫鲁晓夫C戈尔巴乔夫D勃列日涅夫

　　25、二战时期德国武装部队最多是多少

　　A1200万B1000万C500万D800万

　　26、自称为“革命军中马前卒”的资产阶级宣传家是

　　A孙中山B邹容C章炳麟D陈天华

　　27、谁是中国妇女解放运动的先驱

　　A秋瑾B康有为C邓世昌D冯玉祥

　　28、印象派创始人莫奈是哪国画家

　　A法国B德国C中国D英国

　　29、古代印度贵族中，军事贵族属于

　　A首陀罗B婆罗门C吠舍D刹帝利

　　30、第一艘使用蒸汽的轮船

　　A克莱蒙特号B泰坦尼克号C蒙格马利号D斯威夫特号

　　31、18世纪欧洲掀起了什么思想解放运动

　　A宗教改革B文艺复兴C智者运动D启蒙运动

　　32、欧洲经济共同体和欧洲原子能共同体成立在那一年

　　A 1967 B 1945 C1958 D1957

　　33、织田信长在公元那一年被部将明智光秀背叛并被烧死

　　A 1562 B 1560 C 1582 D1581

　　34、红军长征中最后在哪里会师

　　A青海B宁夏C沈阳D甘肃

　　35、我国进入社会主义初级阶段的标志是

　　A第一个五年计划的开始B中华人民共和国成立C三大改造的基本完成D中华人民共和国宪法的制定

　　36、台儿庄的指挥官是谁

　　A陈毅B孙明仁C李宗仁D毛泽东

　　37、《北京条约》是清政府什么时候被迫签订的

　　A 1842 B 1860 C 1752 D 1877

　　38、伊丽莎白一世继位于

　　A1588 B 1558 C 1587 D 1557

　　39、以下四位作为明初的开国功臣，善终的是?

　　A冯胜B徐达C常遇春D汤和

　　40、清朝对西南地区的民族政策改革是哪一项

　　A完全自治B封王设藩C放任自流D改土归流

　　41、下列那位将领不在中国解放军十大元帅之列

　　A：徐向前 B：粟裕 C：聂荣臻D：陈毅

　　42、下列四人中孙中山的妻子是谁

　　A：宋美龄 B：宋祖英 C：宋庆龄 D：宋茜

　　43、吕布被谁杀了

　　A暗杀B曹操C关羽D自爆

　　44、历史上华夏九州，誉为“东方雅典”的?

　　A广州B郑州C苏州D徐州

　　46、伍子胥原本是哪国人

　　A魏国B齐国C韩国D楚国

　　47、在欧洲国家唯一与现今台湾政府有邦交关系

　　A立陶宛B梵蒂冈C保加利亚D冰岛

　　48、罗斯福哪一年就任总统

　　A 1932 B 1937 C 1935 D 1933

　　49、中国的四大石窟，有洛阳龙门石窟，云冈石窟，敦煌莫高窟，还有?

　　A佳丽石窟B麦积山石窟C坛安石窟D黄芝石窟
