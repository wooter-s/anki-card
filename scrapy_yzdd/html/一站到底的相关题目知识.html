　　《一站到底》是江苏卫视继《非诚勿扰》、《非常了得》等一系列金牌栏目后，于2012年3月2日推出的一档全新益智答题类节目，下面是小编为大家提供的一站到底的相关题目知识，大家可以参考下：

　　一站到底的相关题目知识：

　　1、中东哪一城市是三个宗教的圣地? ( D )

　　A、伊斯坦布尔 B、大马士革 C、拿撒勒 D、耶路撒冷

　　2、俄罗斯的斯大林格勒更名为：( A )

　　A、伏尔加格勒 B、列宁格勒 C、戈尔巴乔夫格勒 D、明斯克

　　3、位于西非尖端的最主要港口达喀尔是哪一个国家的首都：( A )

　　C、石油输出国组织(OPEC) D、东南亚国家联盟(ASEAN)

　　4、下列组合中，两地主要产业部门不相一致的是：( D )

　　A、福山——匹兹堡 B、丰田——底特律

　　C、硅谷——九州岛 D、休斯敦——长崎

　　5、有关尼罗河、阿姆河、印度河的下列说法，正确的是：( C )

　　A、都是外流河 B、都流经热带沙漠地区

　　C、都是沿岸地区，重要的灌溉水源 D、都是古代文明的摇篮

　　6、中亚、西亚和北非比较，下列说法不正确的是：( B )

　　A、居民大多信仰伊斯兰教 B、居民都以阿拉伯人为主

　　C、石油是三地重要的矿产 D、农业主要为畜牧业和灌溉农业

　　7、巴西人最喜爱的一种舞蹈是：( B )

　　A、华尔兹 B、桑巴 C、探戈 D、踢踏

　　8、地跨两大洲，首都在西半球的是：( C )

　　A、埃及 B、俄罗斯 C、美国 D、土耳其

　　9、全球第一部国家级的《21世纪议程》诞生于：( D )

　　A、美国 B、巴西 C、日本 D、中国

　　10、在下列东欧诸国中，哪一国家地势低平，四周多强邻，因而在过去两百多年中，疆界多变，并屡遭瓜分? ( D )

　　A、奥地利 B、捷克 C、匈牙利 D、波兰

　　11、进入20世纪以来，申办世博会的国家日益增多，到目前为止，世界博览会共举办了几届：( C )

　　A、28 B、38 C、40 D、41

　　12、2008年第29届夏季奥运会将在我国的北京举办，从气候条件考虑，最佳的比赛时间是：( B )

　　A、9—10月 B、7—8月 C、5—6月 D、3—4月

　　13、世界上最大的区域性贸易集团是：( A )

　　A、欧洲联盟(UN) B、世界贸易组织(WTO)

　　A、塞内加尔 B、象牙海岸(科特迪瓦) C、塞拉利昂 D、毛里塔尼亚

　　14、地中海气候区的雨季通常在：( D )

　　A、春季 B、夏季 C、秋季 D、冬季

　　15、2002年5月20日宣布独立的东帝汶民主共和国位于：( B )

　　A、琉球群岛 B、马来群岛 C、西印度群岛 D、阿留申群岛

　　16、钓鱼岛从何时开始就明确为我国的领土? ( A )

　　A、明朝 B、唐朝 巴元朝 D、宋朝

　　17、我国第一座投入商业运营的核电站是：( A )

　　A、大亚湾核电站 B、泰山核电站 C、岭澳核电站 D、田湾核电站

　　18、世界上最大的白鹤栖息地是在：( A )

　　A、鄱阳湖 B、洞庭湖 C、太湖 D、巢湖

　　19、长江三峡位于最西面的一个峡谷是：( C )

　　A、西陵峡 B、巫峡 C、瞿塘峡 D、三门峡

　　20、当长江发生特大洪水时，下列城市受威胁最大的是：( C )

　　A、重庆 B、郑州 C、武汉 D、杭州

　　21.北方地区与西北地区分界线确定的主导因素是( B )

　　A.地形和地势 /B.季风影响 C.距海远近 D.纬度高低

　　22.下列4国中，本土的地理位置最高的是( D )

　　A.印度 B.中国 C.美国 D.俄罗斯

　　23.下述特定时间里，气温最高的是( A )

　　A.夏季阴天正午 B.春季晴天14时 C.春季晴天正午 D.夏季晴天日出前

　　24.诱发海啸的原因是( C )

　　A.台风 B.龙卷风 C.海底地震 D.大陆地震

　　25.除春秋分日外，地球上凡太阳正在直射的地方必然是( C )

　　A.昼夜平分 B.昼短夜长 C.昼长夜短 D.极昼或极夜

　　26.南回归线附近看不到( D )

　　A.昼夜平分 B.阳光直射 C.日影正南 /D.日影正北

　　27.我国不受寒潮影响的地区是( A )

　　A.雅鲁藏布江谷地 B.海南岛 C.台湾岛 D.汾河谷地

　　28.北极地区寒风凛冽，考虑到当地所处风带和气压带，我国北极科考站营地建筑的门窗应避开( D )

　　A.东南方向 B.西南方向 C.西北方向 /D.东北方向

　　29.做徒步一天往返的野外地理考察，下列比例尺最不适用的是( D )

　　A.1：5000 B.1：1000 C.1：25000 /D.1：100000

　　30.下列地图中能够最直观地表示地形高低起伏状况的是( B )

　　A.等高线图 B.地形剖面图 C.分层设色地形图 D.平面示意图

　　31.“神州”号是( D )

　　A.人造卫星 B.航天飞机 C.国际空间站 D.载人航天实验飞船

　　32.下列城市中属“夏季高温多雨，冬季寒冷干燥气候”的是( B )

　　A.乌鲁木齐 B.北京 C.上海 D.广州

　　33.第27届联合国大会决定把每年的6月5日定为( D )

　　A.爱鸟日 B.节水日 C.植树日 D.环境日

　　34.我国最早出现的种植业位于( A )

　　A.黄河、长江流域 B.松花江流域 C.珠江流域 D.四川盆地

　　35.世界上最大的海湾是( D )

　　A.孟加拉湾 B.墨西哥湾 C.亚丁湾 D.几内亚湾

　　36.按最近距离看，北美洲位于非洲的( B )

　　A.东北方向 B.西北方向 C.西南方向 D.东南方

　　37.“猫搬家、狗狂叫、鸡不进窝、鱼儿惊慌水面跳”是( A )

　　A.地震前兆 B.台风前兆 C.海啸前兆 D.泥石流前兆

　　38.下列重大工程中能缓解东部沿海能源短缺的是( D )

　　A.南水北调 B.青藏铁路 C.西电东送 D.西气东输

　　39.美国国土未跨( D )

　　A.北寒带 B.热带 C.北温带 /D.南温带
