　　1.联合国宣布2004年为“国际大米年”。泰国是世界上重要的稻米出口国，泰国的稻田主要分布在(B)

　　A.湄公河三角洲 B.湄南河三角洲

　　C.伊洛瓦底江平原 D.红河

　　2.世界面积最大的平原是什么平原?(C)

　　A.美索不达米亚平原

　　B.西西伯利亚平原

　　C.亚马逊平原

　　D.东北平原

　　3.世界海拔最高的大洲是什么洲?(D)

　　A.亚洲 B.美洲

　　C.非洲 D.南极洲

　　4.世界火山最多的国家是哪里?(B)

　　A.日本 B.印度尼西亚

　　C.中国 D.美国

　　5.最长的内流河是(B)

　　A.塔里木河 B.伏尔加河

　　C.莱茵河 D.阿姆河

　　6.下列四大高原中，崩塌、滑坡、泥石流等地质灾害发生频繁的是(C)

　　A.青藏高原 B.内蒙古高原

　　C..云贵高原 D.黄土高原

　　7.夏季风强的年份，我国容易出现(D)

　　A.南涝北旱 B.西旱东涝

　　C.西涝东旱 D.南旱北涝

　　8.冬至(12.22)时太阳直射(C)

　　A.23°26′N B.23°26′S

　　C.0° D.66°34′N

　　9.图中A、B表示的大洋分别为(A)

　　A.太平洋、大西洋

　　B.大西洋、印度洋

　　C.北冰洋、太平洋

　　D.印度洋、北冰洋

　　10.下列哪种气候亚洲没有分布(B)

　　A.温带大陆性气候

　　B.温带海洋性气候

　　C.热带雨林气候

　　D.亚热带季风季候

　　11.我国年太阳辐射量最多的地方是(A)

　　A.青藏高原

　　B.云贵高原

　　C.四川盆地

　　D.东南丘陵

　　12.分隔湖北和湖南的湖是指(A)

　　A.洞庭湖

　　B.鄱阳湖

　　C.太湖

　　D.千岛湖

　　13.因四季如春而被称为“春城”的是哪座城市? (A)

　　A.昆明

　　B.洛杉矶

　　C.伦敦

　　D.巴塞罗那

　　14.“石头城”是对我国哪座城市的美称?(B)

　　A.昆明

　　B.南京

　　C.苏州

　　D.成都

　　15. 夏季低温少雨，冬季高温多雨是什么气候的特征?(C)

　　A.亚热带季风气候

　　B.热带草原气候

　　C.地中海气候

　　D.温带季风气候

　　16.世界上最长的河流是(D)

　　A.亚马逊河

　　B.莱茵河

　　C.白尼罗河

　　D.尼罗河

　　17.流域面积最大的河流(A)

　　A.亚马逊河

　　B.莱茵河

　　C.长江

　　D.尼罗河

　　18.俄罗斯属于下列哪个洲(C)

　　A.亚洲 B.北美洲

　　C.欧洲 D.大洋洲

　　19.莱茵河经过几个国家(B)

　　A.5 B.6

　　C.7 D.8

　　20.2008年9月25日21时10分(北京时间)，神舟七号飞船在酒泉卫星发射中心升空。飞行2天20小时27分钟，绕地球飞行45圈后，于9月28日17时37分安全着陆，航天员翟志刚、刘伯明、景海鹏健康出舱。阅读材料回答1~3题。

　　飞船发射时，美国东部地区的夏令时(夏令时比当地区时早一个小时)为 (B)

　　A.9月25日8时10分

　　B.9月25日9时10分

　　C.9月26日5时10分

　　D.9月26日9时10分

　　21.日本所在时区为(A)

　　A.东九区 B.西九区

　　C.东十区 D.西十区

　　22.从上海乘火车到达新疆的乌鲁木齐，沿途所见植被景观的变化规律属于植被的哪种地带性规律(B)

　　A. 海拔 B. 经度

　　C. 纬度 D. 垂直

　　23.植被分布与水热条件密切相关，我国森林地带所对应的地区是( B )

　　A. 干旱地区 B. 湿润地区 C. 半干旱地区 D. 半湿润地区

　　24.温带多雨带的主要降水类型是( C )

　　A、对流雨 B、地形雨 C、锋面雨 D、台风雨

　　25.在气压梯度力和地转偏向力共同作用下，风向最终应是( B )

　　A、风向垂直于等压线 B、风向与等压线之间有一个交角

　　C、风向平行于等压线 D、呈漩涡似的风向

　　26.在反气旋控制下的天气特点一般是( A )

　　A、炎热干燥 B、低温少雨 C、晴朗干燥 D、寒冷干燥

　　27.气压梯度的含义是( D )

　　A、地区之间的气压差 B、单位距离间的气压差

　　C、地势高低之间的气压差 D、垂直高度上的气压差

　　28.影响风速大小的主要因素是( A )

　　A、气压梯度力 B、地转偏向力 C、摩擦力 D、海陆分布

　　广州(113°E)的陈先生到纬度大致相同的夏威夷(156°W)休假，飞机在北京时间5：30从旭日东升的当地机场起飞，降落到夏威夷时正值日落。据此回答29、30题

　　29.陈先生乘坐的飞机大约飞行了( A )

　　A.8小时 B.9小时 C.10小时 D.11小时

　　30.陈先生习惯在每天同时与分别在伦敦和悉尼(15l°E，32°S)留学的两个女儿通过互联网进行交流，已知其在伦敦的女儿当地时间6-9时空闲，在悉尼的女儿当地时间18-21时空闲。则其到达夏威夷后与女儿们联系的时间是( A )

　　A.当地时间22-23时 B.北京时间19-20时 C.当地时间21-22时 D.北京时间20-21时

　　31.世界上不受台风影响的国家和地区是(C )

　　A.亚洲东部 B.亚洲南部

　　C.南美洲西海岸 D.北美洲东海岸

　　32.台风是一种极严重的灾害性天气，下列不属于台风灾害的是( C )

　　A.狂风 B.暴雨 C.寒潮 D.风暴潮

　　33.关于我国自然灾害的叙述，正确的是( B )

　　①我国发生地震的频率不高，影响较小。

　　②江南丘陵降雨充沛，地下水位高，土壤盐碱化十分严重。

　　③滑坡、泥石流多发生在我国西部，地面沉降多发生在我国东部许多大城市。

　　④对植被的破坏和大规模工程活动，是泥石流和滑坡日益频繁的重要原因。

　　A.①② B.③④ C.①③ D.②④

　　34. 我国内蒙古地区的自然带，自西向东依次出现荒漠---草原荒漠---森林草原，其自然带的分布规律是：(B )

　　A.纬度地带性 B.经度地带性 C.垂直地带性 D.非地带性

　　35.各自然带沿着南北方向作有规律的更替，这种现象称为(A )

　　A.纬度地带性 B.经度地带性 C.非地带性 D.垂直地带性

　　36.亚热带常绿阔叶林带分布在(A )

　　A.南北纬25°—35°之间的大陆东部

　　B.南北纬30°—40°之间的大陆西部

　　C.南北纬35°—50°之间的大陆东部

　　D.南北纬50°—70°之间的广大地区

　　37.世界上经度地带性最明显的是在哪个大陆的中纬地区( C )

　　A.北美大陆 B.南美大陆 C.亚欧大陆 D.非洲大陆

　　38.温带森林中，在大陆东、西部都有分布，且具有不同气候特点的自然带是(C )

　　A.亚热带常绿硬叶林带 B.亚热带常绿阔叶林带 C.温带落叶阔叶林带 D.亚寒带针叶林带

　　39.世界亚寒带针叶林分布面积最大的国家是( B )

　　A.美国 B.俄罗斯 C.加拿大 D.中国

　　40.在自然资源的开发利用中，人们必须考虑其是一个相互联系、相互制约、互为环境的资源系统，这主要依据自然资源的( B)

　　A.分布的地域差异性 B.整体性

　　C.不可替代性 D.数量的有限性

　　41.遭受破坏后恢复最困难的自然地带是 ( B )

　　A.北美温带落叶阔叶林带 B.巴西热带雨林带

　　C.中国温带草原带 D.俄罗斯苔原带

　　42.沟通了印度洋和太平洋，被称为海上生命线的海峡是( B )

　　A.霍尔木兹海峡

　　B.马六甲海峡

　　C.德洛克海峡

　　D.莫桑比克海峡

　　43.世界上最长的海峡是(C)

　　A.英吉利海峡

　　B.直布罗陀海峡

　　C.莫桑比克海峡

　　D.马六甲海峡

　　44.大兴安岭的走向是(A)

　　A.东北-西南

　　B.东南-西北

　　C.东西

　　D.南北

　　45.下列天气现象中是由暖锋造成的是( A )

　　A.春雨连绵 B.夏季雷雨 C.秋高气爽 D.丰年瑞雪

　　46.气旋控制下的天气是(A)

　　A.狂风暴雨 B.连绵小雨 C.晴朗干燥 D.多云温和

　　47.岩石圈包括( B )

　　A. 组成地壳的岩石 B. 地壳和地幔顶部 C. 地壳和地幔的大部 D. 地壳和岩浆

　　48.软流层是岩浆的重要发源地，它存在于( B )

　　A. 地壳的下部 B.上地幔的上部 C.下地幔的下部 D. 地核内部

　　49.如图所示，太阳直射(D)

　　A.23°26′N

　　B.23°26′S

　　C.66°34′N

　　D.66°34′S

　　50.关于下列山地雪线的说法,正确的是( D ).

　　A.珠穆朗玛峰北坡纬度比南坡高,对应的雪线高度是北坡低于南坡.

　　B.天山北坡降水多余南坡,雪线高度是北低南高.

　　C.阿尔卑斯山南坡降水多于北坡,雪线南低北高.

　　D.安第斯山南端东西坡纬度相同,对应的雪线东低西高.

　　A.澳大利亚春光袭人

　　B.英国寒风凛冽
